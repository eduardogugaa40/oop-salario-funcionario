# Sistema de Funcionários em Java

Projeto simples em Java para praticar os conceitos de Programação Orientada a Objetos, utilizando classe abstrata, herança e polimorfismo. O sistema possui uma classe abstrata `Funcionario` e duas implementações (`CLT` e `PJ`), cada uma com sua própria regra para cálculo de salário. No `main`, os objetos são tratados de forma genérica através da classe abstrata.
